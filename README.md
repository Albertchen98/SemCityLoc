# SemCityLoc — Project Page

Project page for **SemCityLoc: Aerial 6DoF Localization Using Semantic 3D City Models** (ECCV 2026).

Built from the [Nerfies](https://github.com/nerfies/nerfies.github.io) website template.

## Local preview

```bash
python3 -m http.server 8123
# open http://localhost:8123
```

## Deployment

The site auto-deploys to GitHub Pages on every push to `main` via
`.github/workflows/pages.yml`. After the first push, enable Pages once:

**Repo → Settings → Pages → Build and deployment → Source: GitHub Actions.**

The published URL will be `https://<user>.github.io/<repo>/`.
